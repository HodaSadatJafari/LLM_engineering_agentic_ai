راهنمای راه اندازی سامانه هوشمند سفر

این پروژه یک ایجنت هوشمند است که با استفاده از گراف های تصمیم گیری و ابزارهای جستجوی وب، برنامه ریزی دقیق سفر را برای کاربران انجام می دهد.

ویژگی های فنی:
1. معماری دو مرحله ای (Research & Planning): ابتدا اطلاعات زنده از وب استخراج شده و سپس برنامه ریزی انجام می شود.
2. استفاده از LangGraph: برای مدیریت وضعیت و جریان کاری ایجنتیک.
3. اتصال به AvalAI: بهره گیری از مدل های زبانی پیشرفته برای تولید محتوای فارسی با کیفیت.

مراحل اجرا:

۱. نصب کتابخانه های مورد نیاز:
دستور زیر را در ترمینال اجرا کنید:
pip install -r requirements.txt

۲. تنظیمات کلیدهای API:
یک فایل با نام .env ساخته و مقادیر زیر را در آن قرار دهید:
AVALAI_API_KEY=کلید_شما
TAVILY_API_KEY=کلید_تاویلی_شما

۳. اجرای برنامه:
با استفاده از دستور زیر برنامه را در مرورگر باز کنید:
streamlit run main.py

نکته: برای بخش جستجوی وب، از Tavily API استفاده شده است که اطلاعات زنده را به ایجنت تزریق می کند.
