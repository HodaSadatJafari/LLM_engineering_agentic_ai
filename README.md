# پروژه نهایی: TravelAgent
**ارائه دهنده:** پارسا عابدی نژاد

## معرفی پروژه
پروژه **TravelAgent** یک دستیار هوشمند سفر است که ورودی‌های زیر را از کاربر دریافت می‌کند:
* شهر مقصد
* بودجه سفر
* تعداد روزهای سفر

این اطلاعات در قالب یک قالب متنی به هوش مصنوعی (OpenAI) ارسال می‌شود.

## قابلیت‌ها
* **برنامه‌ریزی مالی:** ارائه پیشنهاداتی برای کنترل هزینه‌ها متناسب با بودجه کاربر.
* **پیشنهاد رستوران و تفریح:** معرفی مکان‌های تفریحی و رستوران‌های مناسب در شهر مقصد.
* **طراحی مدرن:** در این پروژه از هوش مصنوعی برای ساخت UI و دیزاین بهتر کمک گرفته شده است.
