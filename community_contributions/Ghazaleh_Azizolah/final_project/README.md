# Travel Assistant Chatbot ✈️

یک دستیار هوشمند برای برنامه‌ریزی کامل سفر کاربر. این پروژه با استفاده از داده‌های فیک (هتل، پرواز، قطار، اقامتگاه، آب و هوا) و هوش مصنوعی، بهترین گزینه‌های سفر را پیشنهاد می‌دهد.

---

## ساختار پروژه

project_folder/
│
├── main.py # فایل اصلی چت بات
├── mock_data.json # داده‌های فیک: هتل، پرواز، قطار، آب و هوا، اقامتگاه
├── requirements.txt # وابستگی‌ها
├── README.md # این فایل
└── .env.example # نمونه متغیرهای محیطی (API_KEY و ...)




ایجاد محیط مجازی و نصب وابستگی‌ها:
python -m venv venv
venv\Scripts\activate     # ویندوز
pip install -r requirements.txt




ویژگی‌ها

پرسش از کاربر برای اطلاعات سفر (مبدا، مقصد، تاریخ، تعداد مسافران، بودجه، علاقه‌مندی‌ها)

پیشنهاد هتل‌ها با محاسبه هزینه کل بر اساس نوع اتاق و تعداد شب‌ها

بررسی پروازها و قطارها با توجه به بودجه و تاریخ سفر

ارائه پوشاک و وسایل ضروری بر اساس پیش‌بینی آب و هوا

اولویت‌بندی گزینه‌ها بر اساس مقرون به صرفه بودن و علاقه‌مندی‌های کاربر

پشتیبانی از مسافرین زیر 12 سال و ترکیب اتاق‌ها مناسب با تعداد مسافران

داده‌های فیک

هتل‌ها و اتاق‌ها: با قیمت هر نوع اتاق، تعداد تخت سینگل و دبل، و تاریخ‌های رزرو مختلف

پروازها و قطارها: با تاریخ‌ها و قیمت‌ها

پیش‌بینی آب و هوا: برای شهرها و تاریخ‌های مختلف

اقامتگاه‌ها: با شرایط مشابه هتل‌ها

توجه: تاریخ‌ها و موجودی‌ها باید با سفر کاربر همخوانی داشته باشند.

نکات مهم

مطمئن شوید کلید API درست تنظیم شده باشد

تاریخ‌ها و تعداد شب‌ها باید با داده‌های موجود در mock_data.json همخوانی داشته باشند

سیستم به صورت خودکار با توجه به علاقه‌مندی‌ها، بودجه و تعداد مسافران بهترین گزینه‌ها را پیشنهاد می‌دهد

هوش مصنوعی تصمیم می‌گیرد چه ترکیب اتاقی مناسب‌تر است و هزینه کل را محاسبه می‌کند

سیستم توصیه می‌کند که با توجه به بودجه، سفر با هواپیما یا قطار بهتر است
